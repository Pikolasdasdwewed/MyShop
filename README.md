MyShop — учебный интернет-магазин мониторов

Этот проект — учебный шаблон интернет-магазина компьютерных мониторов.
Он содержит панель администратора, авторизацию, CRUD-управление товарами, REST API с полной Swagger-документацией, работу с MySQL через Sequelize и автоматическую генерацию тестовых данных.

Проект полностью готов к запуску на локальной машине без ручного создания базы данных.

Основные возможности:
Авторизация

Вход по email + пароль

Две роли: admin и user

Защита маршрутов через middleware

Сессии на express-session

Управление товарами (мониторами)

Просмотр всех мониторов

Добавление / редактирование / удаление (только admin)

Параметры монитора: бренд, модель, диагональ, разрешение, частота, порты, описание, цена и фото

REST API

GET /api/products

GET /api/products/:id

POST /api/products (admin)

PUT /api/products/:id (admin)

DELETE /api/products/:id (admin)

Swagger UI

API полностью задокументировано по OpenAPI 3.0.

Доступ:
http://localhost:3000/api-docs

Авто-инициализация базы MySQL

При первом запуске приложение само:

создаёт БД MyShop

создаёт таблицы users и products

добавляет аккаунты admin и user

добавляет 9 демо-мониторов

Дополнительно есть скрипт полного сброса:

node reset_database.js

Используемые технологии

Dev:

nodemon — автоперезапуск сервера

Требования
1. Node.js 18+
2. XAMPP или MySQL Server

В XAMPP нужно включить только MySQL.
Базу данных вручную создавать не нужно — проект делает это автоматически.

Установка и запуск
1. Склонировать репозиторий
git clone <ссылка-репозитория>
cd MyShop

2. Установить зависимости
npm install

3. Убедиться, что в корне есть файл .env

Вот пример содержимого (у тебя он уже есть):

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASS=
DB_NAME=MyShop
SESSION_SECRET=change_me
PORT=3000

ADMIN_EMAIL=admin@myshop.local
ADMIN_PASSWORD=123

USER_EMAIL=user@myshop.local
USER_PASSWORD=123

4. Включить MySQL в XAMPP

Запускаем только MySQL Database.

5. Запустить проект
npm start


Приложение создаст БД автоматически и заполнит тестовыми данными.

Доступы для входа
Роль	Email	Пароль
Админ	admin@myshop.local
	123
Пользователь	user@myshop.local
	123
Сброс базы данных

Если нужно пересоздать всё с нуля:

node reset_database.js


После этого снова запустить:

npm start
