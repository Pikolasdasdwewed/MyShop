# MyShop — Магазин мониторов (Node.js + Express + MySQL)

Учебный проект, демонстрирующий работу с:
- Express.js
- Sequelize ORM
- MySQL
- Сессионной авторизацией
- CRUD-операциями для товаров
- Swagger-документацией
- EJS-шаблонами

Проект реализует веб-интерфейс админа, API и защищённый доступ на основе сессий.

---

## Основной функционал

### Авторизация
- Вход в систему (email + пароль)
- Сессия хранится в cookie (`connect.sid`)
- Две роли:
  - **admin** — полный доступ, управление товарами
  - **user** — только просмотр каталога

### Управление товарами (CRUD)
Доступно *только администратору*:
- Создание товара
- Редактирование
- Удаление
- Автоматическая вставка 9 демо-мониторов при пустой базе
